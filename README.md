# Cardiovascular_disease_prediction
This project focuses on predicting cardiovascular disease using a linear regression model and a dataset from Kaggle. By employing sophisticated statistical techniques and machine learning algorithms, I achieved an impressive 85% accuracy in my predictions. This project demonstrates the effectiveness of data-driven insights in early detection and risk assessment of cardiovascular conditions, contributing to advancements in healthcare through predictive modeling.
